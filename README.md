当前年化率为 5%（按 1 SBD = 1 SP 来算）。代理是 Steem Power，但利息每天英国凌晨(UTC)以SBD发放。比如：

```
Thank you for supporting @justyy ! Your 3004.67 SP gives you daily interests of 0.413 SBD (5% APR) - (2018-01-04): 1 STEEM = 5.443 SBD
```
